TOELICHTING

Ik heb deze specialisatie gekozen zodat ik mijn portfolio wat levendiger kan maken door animaties. Om deze reden heb ik me vooral gefocust op visuele libraries, zoals AnimeJS en GSAP.Ik heb besloten om GSAP (Green Sock Animation Platform) te gebruiken voor mijn laatste opdracht voor Advanced Development, omdat het compatibel is met zowat alle browsers, erg snel is en vele opties heeft. Ik had graag meer willen uitwerken, waaronder DrawSVG waarmee ik mijn headers getekend wilde hebben, maar helaas zijn dat betaalde functies van GSAP. Ik heb deze library uitgewerkt in mijn portfolio, zodat de specialisatie ook daadwerkelijk iets toevoegt aan mijn portfolio. Dit heb ik gerealiseerd met standaard animaties, scrub animaties en scroll animaties. Uit de vele mogelijkheden die GSAP biedt, heb ik er een aantal in mijn Portfolio verwerkt en deze kan je bekijken door op de 'portfolio' knop in de header te klikken.

GSAP


GSAP is een snelle, robuuste library waarmee je zonder al teveel code gemakkelijk voorwerpen kunt animeren. GSAP is gemakkelijk onder te knie te krijgen, ik zal even uitleggen hoe het werkt.


                gsap.from('#nslogo', {
                    opacity: 0,
                    duration: 1,
                    rotation: 360,
                    scale: 0.1,
                    delay: 2.5,
                    ease:"expo.out"
                  });

Dit is hoe een standaard GSAP animatie eruit ziet. In dit geval, is het de animatie van mijn logo in de header op mijn Portfolio. Ik zal even uitleggen welk element wat doet.


                gsap.richting('geanimeerde object', {
                    opacity: doorzichtigheid van het object
                    duration: lengte van de animatie
                    rotation: rotatie van het object
                    scale: de grootte van het object
                    delay: de vertraging tot de animatie start
                    ease: hoe de animatie verloopt (verschillende effecten)
                  });

Zoals je kan zien, heeft GSAP dus behoorlijk veel mogelijkheden. Van de richting van de animatie (eindpositie van animatie of juist de beginpositie) tot de manier hoe de animatie verloopt, de mogelijkheden zijn eindeloos. Dit is kort maar krachtig hoe GSAP in dit geval werkt. Er zijn nog veel meer mogelijkheden, maar als ik die allemaal uitwerk, dan zijn we hier nog wel even.
