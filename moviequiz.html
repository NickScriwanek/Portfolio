Inleiding

Om development aan te tonen heb ik een Movie Quiz gemaakt met JavaScript, JQuery en het gebruik van de OMDb API. Ik heb hiervoor gekozen omdat ik nog niet eerder met APIs had gewerkt en ik wilde leren hoe het werkt. Enkel een API vond ik niet leuk genoeg, dus heb ik besloten om er een quiz bij toe te voegen om het wat speelser te maken. De knop onder deze paragraaf linkt naar de werkende applicatie.

Inhoud

De quiz begint met 4 radiobuttons met ieder een array. Iedere radiobutton bevat 7 bekende filmtitels van een apart genre (action, comedy, drama en horror). Vervolgens zijn er if-statements om te zien welke knop is geselecteerd. Nadat de gebruiker het gewenste genre via de radiobutton heeft gekozen, klikt de gebruiker op bevestigen waardoor het geselecteerde genre in de var selectedGenre wordt gezet en de button wordt uitgeschakeld. Vervolgens wordt er een randomNumber gegenereerd door een formule, waarmee een random filmtitel in de gekozen array wordt gekozen. Deze random filmtitel uit de gekozen array wordt in randomMovie gezet. Vervolgens wordt de response van de JSON van de randomMovie uit de API opgehaald. Inmiddels is er ook een randomNumber2 gegenereerd, die bepaalt welke vraag er wordt gesteld. Er zijn namelijk 3 soorten vragen: vragen met een poster, vragen met een plot en vragen met een of meerdere directors. Hierbij wordt natuurlijk de tekst van het tekstvak “question” aangepast aan de inhoud en wordt de poster / director / plot ook gepresenteerd. Ten slotte is er de checkAnswer functie, die wordt opgeroepen als de gebruiker op de knop naast het antwoordveld klikt. Voor ieder correct antwoord krijg je een punt en je hebt 10 pogingen om zoveel mogelijk films correct te raden, daarna krijg je de eindscore te zien en wordt de pagina na 5 seconden opnieuw geladen.

Samenvatting

- Ik heb wiskundige formules gebruikt om random getallen te genereren
- Ik heb API calls gebruikt om filminformatie uit een API te halen
- Ik heb JQuery gebruikt om de values en attributes van een Id te veranderen en de JSON op te vragen.
- Ik heb een hoop getElementById gebruikt om de innerHTML en de textContent te veranderen.

Reflectie

Ik verwachtte dat JavaScript erg lastig zou zijn, aangezien ik er nog geen ervaring mee had. Uiteindelijk viel het wel mee, aangezien het aardig op C# lijkt. Het werken met een API bleek achteraf ook makkelijker dan ik had verwacht, door een aantal handige tutorials die mij op weg hebben geholpen (die bronnenlijst). De volgende keer zou ik multiple choice vragen willen implementeren en ook een API die wat uitgebreider is, aangezien OMDb niet al te veel informatie geeft. Verder zou ik ook eerst de filmtitels op de posters maskeren, aangezien het antwoord nu op de posters staat. Ik ben best trots op het eindresultaat, het werkt zoals ik het voor ogen had en ik snap nu de basis van JavaScript en zelfs nog een beetje JQuery.
